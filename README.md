#### 🌟 WindsurfPoolAPI Highlights

> Multi-account pool proxy for Windsurf — powering 113+ models with enterprise features

- **Multi-Platform Binaries** — macOS (ARM/x64), Linux, Windows standalone executables
- **VS Code / Cursor Extension** — manage the proxy directly from your editor
- **Security** — API key auth, path-redaction sanitizer, identity neutralization
- **Performance** — HTTP/2 session pooling, conversation reuse, stall detection
- **CJK Language Support** — auto-detects Chinese/Japanese/Korean and forces model to respond in kind
- **Docker Ready** — one-command deployment with `docker-compose up`
