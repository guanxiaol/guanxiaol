## Manifesto

I keep four things in mind when I decide what to ship.

1. **AI 红利得是所有人的** —— 不是只属于能付得起最贵 API 的人。
2. **没人因为限额、区域定价或 KYC 用不上前沿模型** —— 那是个 bug，能修。
3. **永远用最强的模型** —— 差工具浪费的不只是时间，是几年。
4. **开源是反垄断最后的武器** —— 几百行 MIT 代码，比一千份反垄断投诉管用。

<sub>(EN) Lowest cost, largest dividend · Nobody left behind · Pick the strongest model, never the cheapest by default · Open source is the last anti-trust weapon.</sub>
